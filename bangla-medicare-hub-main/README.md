# MediCare BD — Hospital Management System

A modern hospital management and patient registration system built for Bangladesh's healthcare sector.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
- Supabase

## Getting Started

### Requirements
- Node.js (LTS version)
- npm

### Steps
```sh
# Clone the repository
git clone https://github.com/YOUR_USERNAME/bangla-medicare-hub.git

# Go to project folder
cd bangla-medicare-hub

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Start dev server
npm run dev
```

Open browser at: http://localhost:8080

## Deployment
```sh
npm run build
```

Deploy the `dist/` folder to any static hosting (Netlify, Vercel, GitHub Pages).
